# Rakth-Chitra

### AI-Assisted Non-Invasive Anemia Risk Screening

> A smartphone-based healthcare solution that enables preliminary anemia risk screening using lower eyelid image analysis — designed especially for rural and low-resource environments.

---

## 📌 Overview

Rakth-Chitra is an AI-assisted mobile application that helps identify potential anemia risk levels using images of the **palpebral conjunctiva (inner eyelid)** captured through a smartphone camera.

Traditional anemia diagnosis often requires:

* blood tests,
* laboratory infrastructure,
* trained medical professionals,
* and physical accessibility.

Rakth-Chitra aims to provide a **non-invasive, affordable, and accessible preliminary screening solution** for underserved communities.

---

## 🚀 Key Features

✅ Lower eyelid image capture
✅ AI-assisted image analysis

✅ Smartphone-based accessibility

        ↓

        ↓

---

# 🧠 How It Works

Rakth-Chitra uses image-processing logic to analyze:

* redness intensity,
* pallor characteristics,
* and color distribution patterns

from the captured lower eyelid image.

The system estimates anemia risk levels based on:

* RGB color extraction,
* redness mapping,
* and image analysis heuristics.

> ⚠️ Note:
> Rakth-Chitra is intended for **preliminary screening assistance only** and does not replace professional medical diagnosis.
